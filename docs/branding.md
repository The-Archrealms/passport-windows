# Branding

`passport-windows` uses the provisional emblem of `The Order of the Archrealms` as its primary application and release mark.

The emblem is built from three elements:

- `the ring`: continuity and durable order
- `the arch`: guardianship and stewardship of inheritance
- `the four-point star`: Truth, Peace, Stewardship, and Knowledge

The vector source for the emblem is included here:

- [the-order-of-the-archrealms-emblem.svg](the-order-of-the-archrealms-emblem.svg)

Release packaging and desktop application assets are generated from the same visual grammar by:

- `tools/release/New-PassportWindowsMsixAssets.ps1`
